# 3. Particionamento e sistemas de arquivos

Para o exemplo de três camadas, um servidor de 256 GB (servidor de banco de dados) com um disco lógico (no RAID10) foi pedido, bem como um servidor de 32 GB (servidor de aplicativos) com um disco lógico (no RAID 1). Ambos os servidores vêm com um sistema de arquivos grande igual ao tamanho total de discos.

Quanto ao armazenamento, o servidor não requer nenhuma etapa de instalação adicional.
